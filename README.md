# Автотесты для «Яндекс.Самокат»

Автотесты на Selenium с использованием Page Object Model для учебного сервиса
[qa-scooter.praktikum-services.ru](https://qa-scooter.praktikum-services.ru/).

## Структура проекта

- `locators/` — локаторы элементов, отдельно для главной страницы и страницы заказа
- `pages/` — классы Page Object (`MainPage`, `OrderPage`, `BasePage`)
- `tests/` — тесты
- `url.py` — URL тестируемого сервиса
- `data.py` — тестовые данные

## Что покрыто тестами

### Аккордеон «Вопросы о важном» (`test_accordion.py`)
Параметризованный тест проверяет раскрытие ответа на каждый из 8 вопросов.

### Оформление заказа (`test_order.py`)
Полный позитивный сценарий заказа самоката:
- Клик по кнопке «Заказать» — обе точки входа (верхняя/нижняя)
- Заполнение формы в два шага
- Проверка модалки об успешном оформлении заказа
- Проверка перехода по лого «Самоката» на главную страницу
- Проверка редиректа по лого Яндекса на Дзен в новой вкладке

Тест параметризован по 2 наборам данных × 2 точкам входа (4 сценария).

## Запуск тестов

```bash
pip install -r requirements.txt
pytest
```

## Allure-отчёт

```bash
pytest --alluredir=allure-results
allure generate allure-results -o allure-report --clean
allure open allure-report
```
